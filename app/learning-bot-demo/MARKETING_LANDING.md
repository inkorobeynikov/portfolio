# LearnWords — Умное изучение иностранных слов

## Что это

LearnWords — это экосистема для эффективного запоминания иностранных слов. Приложение использует научный метод интервального повторения (SM-2), который автоматически подбирает оптимальное время для повторения каждого слова.

---

## Как это работает

### Telegram Bot + Mini App

Всё обучение происходит прямо в Telegram — не нужно устанавливать отдельные приложения.

**Telegram Bot** (@LearnWordsBot):
- Добавление новых слов в словарь
- Ежедневные напоминания о повторении
- Точка входа в приложение

**Telegram Mini App** (WebApp внутри Telegram):
- Полноценный интерфейс для обучения
- Интерактивные карточки слов
- Игры для закрепления
- Статистика и настройки

### Browser Extension

Расширение для браузера позволяет:
- Добавлять новые слова прямо с веб-страниц
- Переводить выделенный текст
- Мгновенно сохранять слова в свой словарь

## Добавление слов

### Процесс добавления через бота

**1. Отправьте слово боту**

Можно отправить:
- Текстом — одно слово или до 10 слов через запятую
- Голосовым сообщением — бот распознает и добавит

Примеры:
```
casa
```
```
libro, sole, tempo, acqua, pane
```

**2. Бот генерирует карточку**

Бот автоматически создаёт полную карточку слова:
- Определение на языке оригинала
- Перевод на родной язык
- 3 примера использования в контексте
- Аудио-произношение

**3. Проверьте и подтвердите**

Бот присылает карточку на проверку. Вы можете:
- ✅ **Подтвердить** — слово добавится в ваш словарь
- ❌ **Отклонить** — если перевод неточный или слово не нужно

Это позволяет контролировать качество словаря и добавлять только нужные слова.

## Функции приложения

### Словарные карточки
- Слово с определением на языке оригинала
- Перевод на родной язык
- Аудио-произношение (нативный спикер)
- 3 примера использования в контексте

### Игры для закрепления

4 типа упражнений:

| Игра | Описание |
|------|----------|
| **Letter Puzzle** | Собери слово из перемешанных букв |
| **Recall Typing** | Напиши слово по его переводу |
| **Audio Recognition** | Услышь слово и напиши его |
| **Fill in the Blank** | Заполни пропуск в предложении |

Игры автоматически подбираются под конкретные слова — если у слова есть аудио, появится Audio Recognition; если есть примеры — Fill in the Blank.

### Сессии обучения

Каждый день приложение формирует персональную сессию:
- Новые слова (введение + первичное закрепление)
- Слова на повторение (по расписанию SM-2)
- Игры для закрепления пройденного

### Настройки

- Количество новых слов в день (5-50)
- Расписание уведомлений (по дням недели)
- Часовой пояс
- Язык интерфейса (RU/EN)

## Интервальное повторение (SM-2)

### Почему это работает

Без повторения мозг забывает информацию по экспоненте:
- Через 20 минут — забыто 40%
- Через 1 час — забыто 55%
- Через 1 день — забыто 75%
- Через 1 неделю — забыто 90%

Это называется «кривая забывания» Эббингауза.

### Как работает SM-2

Алгоритм SuperMemo-2 противодействует забыванию, показывая слово именно в тот момент, когда вы вот-вот его забудете:

**1. Первое повторение** — через 1 день
**2. Второе повторение** — через 3 дня (если вспомнили)
**3. Третье повторение** — через 7 дней
**4. Далее** — интервалы растут: 14, 30, 60 дней...

### Адаптация под вашу память

После каждого повторения вы оцениваете, насколько легко вспомнили слово. Алгоритм учитывает это:

- **Легко вспомнили** → интервал увеличивается быстрее
- **С трудом вспомнили** → интервал увеличивается медленнее
- **Не вспомнили** → слово возвращается в начало

Каждое слово имеет свой «фактор лёгкости» (ease factor), который корректируется с каждым повторением.

### Результат

- Слова переходят из кратковременной в долговременную память
- Вы тратите время только на слова, которые действительно нужно повторить
- Эффективность запоминания выше в 3-5 раз по сравнению с обычной зубрёжкой

## Аудио

Каждое слово включает профессиональную озвучку:
- Само слово
- Перевод
- Все примеры предложений

Аудио генерируется нейросетями с естественным произношением носителя языка.
